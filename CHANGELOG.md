## [0.0.2] - 2022-02-27

### Changed

## [0.0.1] - 2022-02-26

### Added

- `yacht add` - add a Laravel Sail project to Yacht
- `yacht start` - start the Yacht service
- `yacht stop` - stop the Yacht service
